Generating/Migrating Models

$ bin/rails generate model RequestHandler column_two:string column_three:text


#this will create several files, but let's begin by examining the new migration file,


#a script file which modifies the structure of any given database


$ cat db/migrate/<timestamp>_create_request_handlers.rb


class CreateRequestHandlers < ActiveRecord::Migration[7.0]


 def change


 create_table :request_handlers do |t|


#the create_table method defines the structure of our request_handlers table


#the first column id is automatically generated and acts as a simple row counter


#MySQL equivalent: CREATE TABLE request_handlers (id INT PRIMARY KEY AUTO_INCREMENT);


  t.string :column_two


  t.text :column_three


#the next two columns we defined in our generate model command

#MySQL equivalent: ALTER TABLE request_handlers ADD column_two VARCHAR ...


  t.timestamps


#the t.timestamps method will add two columns, created_at and updated_at


#which will be maintained by rails


 end


 end


end


$ bin/rails db:migrate

Modifying Actions

$ nano app/controllers/request_handlers_controller.rb

 def index

 @request_handlers = RequestHandler.all

 end
